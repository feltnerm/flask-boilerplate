About Webπ?

Webπ is a collection of web applications that can be used by developers to easily get started on there project. Go ahead, check it out!

Three Core Principles

Webπ is built on three core principles:

  • Simple: Do one thing, do it well.
  • Speedy: It's a tool for rapid development with best practices.
  • Extensible: It provides ways to add more functionality in a snap!

Docs & Support

The easiest way to really get started with Webπ is to check dive in. There is not much documentation or support as of now. If you have any questions, thoughts, concerns or feedback, please don't hesitate to email me at feltnerm@uwplatt.edu.
